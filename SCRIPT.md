# Lit Protocol Animation Script & Storyboard

**Duration:** 60-90 seconds
**Target:** Developers and crypto users
**Key Message:** Self-custodial automation via simple API

---

## Scene Breakdown

### Scene 1: Hook / Problem (10 seconds)
**Visual:**
- Dark background with scattered wallet icons/notifications
- Flashing "Sign transaction" prompts appearing rapidly
- User icon looking overwhelmed

**Text/Narration:**
> "Managing crypto requires constant attention..."
> "Every action needs manual approval"

**Animation:**
- Multiple transaction signing prompts appearing
- Clock ticking, user frustrated
- Build tension with quick cuts

---

### Scene 2: Solution Intro (12 seconds)
**Visual:**
- Lit Protocol logo/wordmark appears
- Clean, calming transition
- Single click animation

**Text/Narration:**
> "Lit Protocol: Self-Custodial Automation"
> "Create an agent wallet with one click"

**Animation:**
- Logo fade in with glow effect
- Cursor clicking once
- Agent wallet materializing

---

### Scene 3: How It Works - API Integration (18 seconds)
**Visual:**
- Split screen showing:
  - Left: Wallet/dApp interface
  - Right: Agent wallet
- API connection lines between them
- Code snippets appearing

**Text/Narration:**
> "Any wallet or dApp can install automations via simple API"
> "Both your app and agent interact through the same API"
> "Perfect for building with Claude Code"

**Animation:**
- API endpoints connecting
- JSON/code elements flowing
- Automation rules being set
- Claude Code logo/mention

---

### Scene 4: Agent in Action (15 seconds)
**Visual:**
- Agent wallet performing multiple operations
- Crypto transactions executing automatically
- DeFi operations, swaps, transfers happening
- All without manual intervention

**Text/Narration:**
> "Your agent executes any crypto transaction automatically"
> "DeFi, transfers, complex operations - all automated"

**Animation:**
- Transaction confirmations appearing smoothly
- Multiple chains/protocols shown
- Smooth, efficient visual flow

---

### Scene 5: Self-Custodial Control (15 seconds)
**Visual:**
- User icon maintaining connection to assets
- Withdrawal flow visualization
- Lock/unlock mechanism showing user control
- Safety/security shields

**Text/Narration:**
> "Always self-custodial"
> "Withdraw your assets anytime"
> "You maintain full control"

**Animation:**
- User pulling assets back instantly
- Security indicators glowing
- Trust and control emphasized

---

### Scene 6: Call to Action (8 seconds)
**Visual:**
- Lit Protocol wordmark
- Website URL
- "Get Started" button/link
- Social links

**Text/Narration:**
> "Build self-custodial automation today"
> "litprotocol.com"

**Animation:**
- Logo final position
- URL and links fade in
- Clean, professional close

---

## Technical Details to Emphasize

1. ✅ **Self-custodial** - User always in control
2. ✅ **One-click setup** - Agent wallet creation
3. ✅ **Universal API** - Any wallet/dApp can integrate
4. ✅ **Simple for developers** - Ideal with Claude Code
5. ✅ **Withdraw anytime** - User maintains custody
6. ✅ **Any crypto operation** - Full flexibility

## Color Palette
- Primary: Lit Protocol brand colors (to extract from logos)
- Accent: Purple/blue gradient (crypto/tech feel)
- Background: Dark (#0a0a0a to #1a1a1a)
- Highlights: Orange (from Lit logomark)
- Success: Green (#10b981)
- Code blocks: Monospace with syntax highlighting

---

## Typography
- Headers: Bold, modern sans-serif
- Body: Clean sans-serif
- Code: Monospace (JetBrains Mono or Fira Code style)
- Logo: Lit Protocol brand font

---

## Animation Style
- Smooth, professional transitions
- Not too fast - give viewers time to absorb
- Modern, clean aesthetic
- Emphasis on simplicity and ease of use
- Technical but approachable
